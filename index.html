Murmur

Python project that triangulates the locations of sounds overtime to generate heat-maps and find quiet locations in a building. The module performs sound analysis on NumPy data, and uses vector addition to determine the positions of the sounds with a Raspberry Pi. Information is sent and viewable on the web application.

AutoML visual recognition project designed to assist those with visual impairments.
